Expose informations from your Wordpress, in JSON format, to allow you to create centralized dashboards (not included ...)

== Description ==

The basic idea is to be able to build a dashboard centralizing information from several Wordpress, and thus monitor them all at once ...

Warning : this plugin is mostly technical...

== Installation ==

1. Go to your admin area and select Plugins â†’ Add New from the menu.
2. Search for "JSON Dashboard Infos" (by Richard Carlier).
3. Click install.
4. Click activate.
6. Navigate to Settings > JSON Dashboard Infos.
7. Select options you want, and validate
8. Navigate to Settings > Permalink, and just validate (if needed)
